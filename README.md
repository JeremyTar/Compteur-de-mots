# Compteur-de-mots :black_nib:

Compteur de mots en Typescript 

:point_right: [Demo here](https://www.quoteverba.sayadini.fr) :point_left:

## Stack :package:
- Package manager : NPM
- CSS organisation : Sass 

## Run it :computer:

- Pull project 
```
git pull https://github.com/JeremyTar/Compteur-de-mots.git
```
- Use 
```
npm install
```
- watch sass file
```
npm run sass
```
- Build typescript
```
npm run build
```
- Open html file
